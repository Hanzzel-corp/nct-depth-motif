# How to Interpret Results

Guide for reading result files generated by NCT experiments.

---

## Four Models Evaluated

Each execution trains and evaluates four weight table variants:

| Model | Description | Use Case |
|-------|-------------|----------|
| `motif_survival` | Full weights with shrinkage | Main model |
| `motif_survival_pos_only` | Only weights > 0 | Identify "protective" motifs |
| `motif_survival_neg_only` | Only weights < 0 | Identify "rupture" motifs |
| `motif_survival_binary` | Sign only (+1/-1/0) | Minimum information test |

## Reference Results

Expected values from successful runs on NYU Depth V2:

### Grouped numeric block split (30 runs)

3 alphas × 10 seeds, blocks of 50 frames, 256 random baselines:

| Model | p_F1 | p_AUC | p_Spearman | < 0.05 in |
|-------|------|-------|------------|-----------|
| `motif_survival` | 0.0039 | 0.0079 | 0.0039 | 30/30 |
| `motif_survival_binary` | 0.0039 | 0.0039 | 0.0039 | 30/30 |
| `motif_survival_pos_only` | 0.0042 | 0.0926 | 0.0525 | F1 30/30, AUC 1/30 |
| `motif_survival_neg_only` | 0.1540 | 0.2432 | 0.0660 | 0/30 |

**Magnitudes:** ΔAUC ≈ +0.004, ΔF1 ≈ +0.005 vs random

## Reading p-values

The p-value indicates the probability that the result is due to chance.

| Range | Interpretation |
|-------|----------------|
| **p < 0.0039** | Maximum significance. With 256 randoms, model exceeds ALL possible random runs |
| **p < 0.01** | Highly significant. Strong evidence against H0 |
| **p < 0.05** | Significant (classical). Standard evidence against H0 |
| **p = 0.05-0.10** | Trend. Requires more data |
| **p > 0.10** | Not significant. Indistinguishable from random |

**Formula:** `p = (1 + n_better_random) / (1 + N_random)`

## Key Finding: Binary = Full

### Observation

`motif_survival_binary` achieves the **same significance** as the full `motif_survival` model (sometimes higher AUC).

### Interpretation

| Aspect | Conclusion |
|--------|------------|
| **Useful information** | Is in the **direction** of association (+/-) |
| **Magnitude** | Adds noise rather than discriminative value |
| **Implication** | NCT quantization captures real geometric structure |

### Why This Happens

The direction indicates:
- **Positive weight**: motif associated with survival (no rupture)
- **Negative weight**: motif associated with rupture

The exact magnitude depends on shrinkage and sample size, introducing unnecessary variability.

## Effect Sizes

### Observed Effect Size

| Metric | Improvement vs Random | Interpretation |
|--------|----------------------|----------------|
| ΔAUC | ~+0.004 | Small but consistent |
| ΔF1 | ~+0.005 | Small but consistent |

### Context

```
NCT effect:   +0.004 in AUC  (p < 0.01)
Canny effect: +0.150 in AUC  (estimated, not directly measured)
```

**Verdict:**
- ✅ Statistically significant (rejects H0)
- ⚠️ Practically modest (does not replace classical detectors)
- 💡 Complementary: corrects delta in ambiguous zone

---

## Interpretation Limits

### What CAN Be Concluded

- NCT motifs contain statistically significant information about 3D discontinuities
- Weight direction is more informative than its magnitude
- The method generalizes within NYU Depth V2 (scene LOO)

### What CANNOT Be Concluded (yet)

| Claim | Reason |
|-------|--------|
| "Superior to Canny/Sobel/HED" | Not directly compared in same benchmark |
| "Generalizes to ScanNet/KITTI" | Only tested on NYU Depth V2 |
| "NCT is the optimal quantization" | Other quantizations not systematically explored |
| "Production-ready" | Marginal improvements over classical baseline |

## Reproduction Checklist

To confirm your results are consistent:

- [ ] `p_sp`, `p_auc`, `p_f1` match (or are very similar to) reference
- [ ] `motif_survival_binary` is competitive with `motif_survival`
- [ ] All test models outperform `random_motif_mean`
- [ ] Effect concentrated in AMBIGUOUS_ONLY zone
- [ ] Weight signs are consistent across runs (e.g., `(+,+,-)` is usually negative)

## Reading Example

Typical fragment from `summary.csv`:

```csv
model,alpha,p_sp,p_auc,p_f1
motif_survival,0.03,0.0039,0.0078,0.0039
motif_survival_binary,0.03,0.0039,0.0039,0.0039
random_motif_mean,0.03,0.2500,0.4500,0.3200
```

**Reading:** Both NCT models achieve maximum/high significance, while the random average does not exceed expected chance (p ≈ 0.5).
